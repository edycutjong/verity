# Security Policy

## Supported Versions

The following versions of this project are currently supported with security updates:

| Version | Supported |
| --- | --- |
| v1.0.x | :white_check_mark: |
| < v1.0.0 | :x: |

## Reporting a Vulnerability

We take the security of this project seriously. If you discover a security vulnerability, please report it privately to ensure the safety of our users.

To report a vulnerability:
1. Send an email to **edy.cu@live.com**.
2. Include a detailed description of the vulnerability, steps to reproduce, and any potential impact or proof of concept.

We will acknowledge receipt of your report within 48 hours and work with you to resolve the issue as quickly as possible. Please do not open public GitHub issues for security vulnerabilities.
